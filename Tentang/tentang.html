HTML 5

HTML, atau HyperText Markup Language, adalah bahasa markup yang digunakan untuk membuat struktur dasar sebuah halaman web. Dengan HTML, Anda dapat menentukan bagaimana konten pada halaman web Anda akan diorganisir dan ditampilkan di browser. Bahasa ini sangat cocok untuk Anda yang belum pernah koding sebelumnya.

CSS 3

CSS, atau Cascading Style Sheets, adalah bahasa yang digunakan untuk memberikan gaya atau tata letak (styling) pada elemen-elemen HTML. Dengan menggunakan CSS, Anda dapat mengendalikan tampilan visual dari halaman web Anda secara terpisah dari struktur HTML. Ini memungkinkan pemisahan antara konten dan presentasi. Bersamaan dengan HTML, bahasa ini sangat cocok untuk pemula.

JavaScript

JavaScript adalah bahasa pemrograman yang dapat digunakan untuk menambahkan interaktivitas dan dinamika pada halaman web. Dengan JavaScript, Anda dapat mengontrol perilaku halaman, merespons interaksi pengguna, memanipulasi data, dan melakukan banyak fungsi lainnya secara dinamis di sisi klien (di browser pengguna). Bahasa ini cocok untuk dipelajari setelah Anda menguasai HTML dan CSS.

Bootstrap

Bootstrap adalah kerangka kerja (framework) front-end yang populer untuk pengembangan web. Dikembangkan oleh Twitter, Bootstrap menyediakan kumpulan alat dan gaya yang dapat digunakan untuk mempercepat dan menyederhanakan proses pengembangan halaman web yang responsif dan menarik. Bila Anda susah mempelajari CSS, Bootstrap ini sangat direkomendasikan untuk Anda pelajari agar koding CSS Anda bisa lebih mudah.
